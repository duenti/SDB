CEvADA

CEvADA is a database of co-evolutionary clusters in conserved protein domains. The current version is based in the Pfam 32.0 and all the data was calculated using CONAN. At the moment, CEvADA contains {{ sdb_n }} entries, comprising {{ sdb_fraction }}% of the protein families available in Pfam. More details about the calculation algorithms and examples of applications can be found here.

Cite Us

CEvADA is not published yet, although the manuscript is in preparation. For now, please cite the software that was used to calculate the data:

Fonseca, N. J., Afonso, M. Q. L., Carrijo, L., & Bleicher, L. (2020). CONAN: a web application to detect specificity determinants and functional sites by amino acids co-variation network analysis. Bioinformatics.
